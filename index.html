about restudio

Restudio was founded in January 2025. We are passionate about game development and creating useful tools for developers and creators.

Our main projects include Pixmancer, a pixel art companion tool, and Wetbox Game Engine, a lightweight framework for indie developers.

We also build tools like Restudio Script Editor and Motion Studio X to help streamline creative workflows. Everything we make is driven by curiosity and the desire to make game development more accessible.
